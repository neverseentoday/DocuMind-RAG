# DocuMind-RAG
Secure RAG-based PDF Q&amp;A system enabling authenticated users to upload documents and ask natural language queries. Uses FAISS for vector search, Sentence-Transformers for embeddings, and a local LLM via Ollama to deliver accurate, context-aware, privacy-preserving responses.
